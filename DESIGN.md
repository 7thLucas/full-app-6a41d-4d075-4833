# Big II Consulting AI OS — Design System

## Brand Colors
- Primary Dark: #0F1E3C (deep navy) — backgrounds, headers, dark panels
- Gold Accent: #C8901A — primary actions, highlights, AI governor accents
- Gold Light: #F59E0B — hover states, secondary gold
- White: #FFFFFF — content surfaces, cards
- Slate 50: #F8FAFC — light panel backgrounds
- Slate 400: #94A3B8 — secondary text, labels
- Slate 600: #475569 — body text on light backgrounds

## Governor Color Coding
- Ω-Praxis: Gold (#C8901A)
- Lyra: Emerald (#6EE7B7 / #059669)
- Kael: Blue (#93C5FD / #3B82F6)
- ARAI: Rose (#FCA5A5 / #EF4444)

## Typography
- Display/Headline: font-black (900 weight), tight tracking (-0.02em), deep navy
- Labels: 0.6rem, ALL CAPS, letter-spacing 0.2em, gold or slate-400
- Body: 0.8–0.9rem, slate-600, line-height 1.6

## Elevation & Cards
- Dark cards: bg #0F1E3C, no border, rounded-xl/2xl
- Light cards: bg #F8FAFC, border 1px #E2E8F0, rounded-xl
- Featured/accent cards: left border 3px gold, bg white
- Gold-tinted: bg #FEF6E4, border #F59E0B

## Navigation
- Sidebar: deep navy (#0F1E3C), white icons/text, gold active state
- Top bar: white, subtle bottom border

## Dashboard Layout
- Sidebar navigation (left, fixed)
- Main content area (scrollable)
- AI Activity Feed widget
- Revenue Snapshot widget
- System Health widget (Kael)
- Integrity Alerts widget (Lyra)

## Key UI Screens to Build
1. Dashboard — AI activity feed, revenue snapshot, lead pipeline overview, system health
2. White-Label Onboarding Wizard — step-by-step: brand setup → PayPal connect → portal preview → go live
3. Consulting Engine — start new engagement, view deliverables, client portal
4. Lead Engine — ARAI pipeline overview, outreach status, qualification scores
5. Billing — subscription management, usage tracking, invoice history
6. Settings — account, team, integrations, API keys

## Tone
Commanding, sovereign, enterprise-grade. Think control room, not dashboard. Every element should feel like infrastructure, not software.
